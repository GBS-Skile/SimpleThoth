# 각 줄의 첫 번째 단어는 명령어다.
# 이를테면 2번째 줄의 "명령어"는 "#"이며, 이는 뒤의 내용을 무시하는 역할을 시킨다.
# 명령어는 whitespace로 구분되어야만 한다. 즉 "#주석"은 유효한 주석이 아니다.

    # 앞 뒤 공백은 무시된다.    



# STATE 명령어는 이 이후로 작성되는 명령문이 
STATE Default
> 안녕하세요! 전 터리예요.
> 사용자님의 이름은 무엇인가요?



STATE InputName
# msg : 사용자 입력 그대로 갖다 붙이기
# Dialog_어쩌구
# User_어쩌구
> {msg}님 맞으신가요?
SET context_User_#name "{msg}"
: 김철수



STATE NameCheck
> 죄송하지만 말씀을 이해하지 못 했어요.
> {User_#name}님 맞으신가요?
GOTO NameCheck
< 응
  > 그렇군요.
  > 반가워요, {User_#name}님!
  > 요즘은 잘 지내세요?
  GOTO StatusCheck
< 아니
  > 그렇군요...
  > 다시 이름을 말씀해 주시겠어요?
  GOTO InputName



STATE StatusCheck
> 아하 그러시군요.
: 그럭저럭
: 잘 못 지내



STATE End
> 현재는 여기까지 구현된 상태랍니다.
> 다시 시작하려면 '초기화'라고 말해주세요.
GOTO End
< 초기화
  > 기억을 잃는 중입니다...
  GOTO Default



STATE?
> 음... 뭔가 이상하네요.
> 처음부터 다시 대화를 시작해보죠.
GOTO Default
